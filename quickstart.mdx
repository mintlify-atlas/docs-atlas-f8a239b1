# Quickstart Guide

This guide will help you make your first API request and scrape a website in minutes.

## Get Your API Key

<Steps>
  <Step title="Sign up for Firecrawl">
    Go to [firecrawl.dev](https://firecrawl.dev) and create a free account.
  </Step>
  
  <Step title="Generate an API key">
    Navigate to your dashboard and generate a new API key. Your API key will start with `fc-`.
  </Step>
  
  <Step title="Save your API key">
    Store your API key securely. You'll need it for all API requests.
  </Step>
</Steps>

<Warning>
Never commit your API key to version control. Use environment variables to keep it secure.
</Warning>

## Make Your First Request

Let's scrape a website and get clean markdown output:

<CodeGroup>
```bash cURL
curl -X POST 'https://api.firecrawl.dev/v2/scrape' \
  -H 'Authorization: Bearer fc-YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "url": "https://firecrawl.dev",
    "formats": ["markdown", "html"]
  }'
```

```python Python
from firecrawl import Firecrawl

app = Firecrawl(api_key="fc-YOUR_API_KEY")

# Scrape a single URL
doc = app.scrape("https://firecrawl.dev", formats=["markdown", "html"])
print(doc.markdown)
print(doc.metadata)
```

```javascript JavaScript
import Firecrawl from '@mendable/firecrawl-js';

const app = new Firecrawl({ apiKey: 'fc-YOUR_API_KEY' });

// Scrape a single URL
const scrapeResponse = await app.scrape('https://firecrawl.dev', {
  formats: ['markdown', 'html'],
});
console.log(scrapeResponse.markdown);
console.log(scrapeResponse.metadata);
```
</CodeGroup>

### Response

You'll receive a JSON response with the scraped content:

```json
{
  "success": true,
  "data": {
    "markdown": "# Firecrawl\n\nTurn websites into LLM-ready data...",
    "html": "<!DOCTYPE html><html>...",
    "metadata": {
      "title": "Firecrawl - The Web Data API for AI",
      "description": "The web crawling, scraping, and search API for AI.",
      "sourceURL": "https://firecrawl.dev",
      "statusCode": 200
    }
  }
}
```

## Install an SDK

For production applications, we recommend using one of our official SDKs:

<Tabs>
  <Tab title="Python">
    Install the Python SDK:
    
    ```bash
    pip install firecrawl-py
    ```
    
    Basic usage:
    
    ```python
    from firecrawl import Firecrawl
    
    app = Firecrawl(api_key="fc-YOUR_API_KEY")
    
    # Scrape a website
    doc = app.scrape("https://firecrawl.dev", formats=["markdown"])
    print(doc.markdown)
    
    # Crawl a website (automatically waits for completion)
    docs = app.crawl("https://docs.firecrawl.dev", limit=50)
    for doc in docs.data:
        print(doc.metadata.source_url, doc.markdown[:100])
    
    # Search the web
    results = app.search("best web scraping tools 2024", limit=10)
    print(results)
    ```
  </Tab>
  
  <Tab title="JavaScript">
    Install the Node.js SDK:
    
    ```bash
    npm install @mendable/firecrawl-js
    ```
    
    Basic usage:
    
    ```javascript
    import Firecrawl from '@mendable/firecrawl-js';
    
    const app = new Firecrawl({ apiKey: 'fc-YOUR_API_KEY' });
    
    // Scrape a website
    const doc = await app.scrape('https://firecrawl.dev', { formats: ['markdown'] });
    console.log(doc.markdown);
    
    // Crawl a website (automatically waits for completion)
    const docs = await app.crawl('https://docs.firecrawl.dev', { limit: 50 });
    docs.data.forEach(doc => {
        console.log(doc.metadata.sourceURL, doc.markdown.substring(0, 100));
    });
    
    // Search the web
    const results = await app.search('best web scraping tools 2024', { limit: 10 });
    results.data.web.forEach(result => {
        console.log(`${result.title}: ${result.url}`);
    });
    ```
  </Tab>
  
  <Tab title="Go">
    Install the Go SDK:
    
    ```bash
    go get github.com/mendableai/firecrawl-go
    ```
    
    Check out the [Go SDK documentation](https://github.com/mendableai/firecrawl-go) for usage examples.
  </Tab>
  
  <Tab title="Rust">
    Install the Rust SDK:
    
    ```bash
    cargo add firecrawl
    ```
    
    Check out the [Rust SDK documentation](/sdks/rust) for usage examples.
  </Tab>
</Tabs>

## Extract Structured Data

One of Firecrawl's most powerful features is extracting structured data using schemas:

<CodeGroup>
```python Python
from firecrawl import Firecrawl
from pydantic import BaseModel

app = Firecrawl(api_key="fc-YOUR_API_KEY")

class CompanyInfo(BaseModel):
    company_mission: str
    is_open_source: bool
    is_in_yc: bool

result = app.scrape(
    'https://firecrawl.dev',
    formats=[{"type": "json", "schema": CompanyInfo.model_json_schema()}]
)

print(result.json)
# Output: {"company_mission": "Turn websites into LLM-ready data", "is_open_source": true, "is_in_yc": true}
```

```javascript JavaScript
import Firecrawl from '@mendable/firecrawl-js';
import { z } from 'zod';

const app = new Firecrawl({ apiKey: 'fc-YOUR_API_KEY' });

const schema = z.object({
  company_mission: z.string(),
  is_open_source: z.boolean(),
  is_in_yc: z.boolean(),
});

const result = await app.scrape('https://firecrawl.dev', {
  formats: [{ type: 'json', schema }]
});

console.log(result.json);
// Output: {company_mission: "Turn websites into LLM-ready data", is_open_source: true, is_in_yc: true}
```
</CodeGroup>

<Note>
You can also extract data without a schema using a simple prompt:

```python
result = app.scrape(
    'https://firecrawl.dev',
    formats=[{"type": "json", "prompt": "Extract the company mission"}]
)
```
</Note>

## Crawl an Entire Website

To scrape multiple pages from a website:

<CodeGroup>
```bash cURL
curl -X POST 'https://api.firecrawl.dev/v2/crawl' \
  -H 'Authorization: Bearer fc-YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "url": "https://docs.firecrawl.dev",
    "limit": 100,
    "scrapeOptions": {
      "formats": ["markdown"]
    }
  }'
```

```python Python
from firecrawl import Firecrawl
from firecrawl.types import ScrapeOptions

app = Firecrawl(api_key="fc-YOUR_API_KEY")

# Crawl waits for completion automatically
crawl_result = app.crawl(
    'https://docs.firecrawl.dev',
    limit=100,
    scrape_options=ScrapeOptions(formats=['markdown'])
)

for doc in crawl_result.data:
    print(f"{doc.metadata.source_url}: {doc.markdown[:100]}")
```

```javascript JavaScript
import Firecrawl from '@mendable/firecrawl-js';

const app = new Firecrawl({ apiKey: 'fc-YOUR_API_KEY' });

// Crawl waits for completion automatically
const crawlResponse = await app.crawl('https://docs.firecrawl.dev', {
  limit: 100,
  scrapeOptions: { formats: ['markdown'] },
});

crawlResponse.data.forEach(doc => {
  console.log(`${doc.metadata.sourceURL}: ${doc.markdown.substring(0, 100)}`);
});
```
</CodeGroup>

<Note>
The SDKs automatically handle polling for crawl completion. With the REST API, you'll need to poll the status endpoint manually.
</Note>
