---
title: 'Get token usage'
description: 'Get remaining tokens for Extract feature'
---

## Endpoint

<CodeGroup>
```http
GET /v1/team/token-usage
```
</CodeGroup>

## Authentication

This endpoint requires authentication using a Bearer token. Include your API key in the Authorization header:

```
Authorization: Bearer YOUR_API_KEY
```

## Response

<ResponseField name="success" type="boolean">
  Indicates whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  Token usage information
  <Expandable title="properties">
    <ResponseField name="remaining_tokens" type="number">
      Number of tokens remaining for the team
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X GET https://api.firecrawl.dev/v1/team/token-usage \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

```python Python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key='YOUR_API_KEY')

usage = app.get_token_usage()

print(usage)
```

```javascript JavaScript
import FirecrawlApp from '@mendable/firecrawl-js';

const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

const usage = await app.getTokenUsage();

console.log(usage);
```
</CodeGroup>

## Response Example

```json
{
  "success": true,
  "data": {
    "remaining_tokens": 1000
  }
}
```

## Error Responses

<ResponseField name="404" type="object">
  **Not Found** - Could not find token usage information.
  
  ```json
  {
    "success": false,
    "error": "Could not find token usage information"
  }
  ```
</ResponseField>

<ResponseField name="500" type="object">
  **Server Error** - Internal server error while fetching token usage.
  
  ```json
  {
    "success": false,
    "error": "Internal server error while fetching token usage"
  }
  ```
</ResponseField>
